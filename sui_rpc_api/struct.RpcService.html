pub fn health_check(&self, threshold_seconds: Option<u32>) -> Result<()>

Perform a simple health check on the service.

The threshold, or delta, between the server’s system time and the timestamp in the most recently executed checkpoint for which the server is considered to be healthy.

If not provided, the server will be considered healthy if it can simply fetch the latest checkpoint from its store.
